# 什么是 Claude？

Claude 是由 Anthropic 开发的 AI 助手，它可以帮你写代码、回答问题、分析文档。简单来说，它就像一个 24 小时在线的编程搭档。

## Claude 能做什么？

- **写代码**：告诉它你想要什么功能，它就能生成代码
- **改代码**：把现有代码发给它，它能帮你优化、修 bug
- **解释代码**：看不懂的代码？Claude 可以用大白话解释给你听
- **写文档**：README、注释、API 文档都能搞定

## Claude Code 是什么？

Claude Code 是 Anthropic 推出的命令行工具，让你直接在终端里和 Claude 对话。它能直接读写你项目里的文件，比网页版更方便。

```bash
# 安装 Claude Code
npm install -g @anthropic-ai/claude-code

# 启动
claude
```

> **提示**：Claude Code 会自动识别你当前目录下的项目结构，所以建议在项目根目录下启动。

## Claude 和其他 AI 有什么区别？

| 特点 | Claude | 其他 AI |
|------|--------|---------|
| 上下文长度 | 超长上下文，能理解整个项目 | 通常较短 |
| 代码质量 | 高质量，遵循最佳实践 | 参差不齐 |
| 安全性 | 内置安全机制 | 因产品而异 |
