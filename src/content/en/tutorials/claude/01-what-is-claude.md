# What is Claude?

Claude is an AI assistant developed by Anthropic. It can help you write code, answer questions, and analyze documents. Think of it as a 24/7 coding partner.

## What Can Claude Do?

- **Write code**: Tell it what you want, and it generates code
- **Fix code**: Send existing code, and it can optimize or debug it
- **Explain code**: Don't understand some code? Claude explains it in plain English
- **Write docs**: README files, comments, API documentation — all handled

## What is Claude Code?

Claude Code is a command-line tool from Anthropic that lets you chat with Claude directly in your terminal. It can read and write files in your project, making it more convenient than the web version.

```bash
# Install Claude Code
npm install -g @anthropic-ai/claude-code

# Start
claude
```

> **Tip**: Claude Code automatically detects your project structure, so it's best to start it in your project's root directory.

## How is Claude Different from Other AIs?

| Feature | Claude | Other AIs |
|---------|--------|-----------|
| Context length | Very long, can understand entire projects | Usually shorter |
| Code quality | High quality, follows best practices | Varies |
| Safety | Built-in safety mechanisms | Depends on product |
